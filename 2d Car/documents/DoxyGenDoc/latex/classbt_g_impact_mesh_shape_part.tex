\doxysubsection{Detailed Description}
This class manages a sub part of a mesh supplied by the bt\+Striding\+Mesh\+Interface interface. 


\begin{DoxyItemize}
\item Simply create this shape by passing the bt\+Striding\+Mesh\+Interface to the constructor \doxylink{classbt_g_impact_mesh_shape_part}{bt\+GImpact\+Mesh\+Shape\+Part}, then you must call \doxylink{classbt_g_impact_shape_interface_acb26c2d7a2aecabd06b996b72b848492}{update\+Bound()} after creating the mesh
\item When making operations with this shape, you must call {\bfseries{lock}} before accessing to the trimesh primitives, and then call {\bfseries{unlock}}
\item You can handle deformable meshes with this shape, by calling \doxylink{classbt_g_impact_shape_interface_ac430754948ac92d6f70b81e88416c96a}{post\+Update()} every time when changing the mesh vertices. 
\end{DoxyItemize}
