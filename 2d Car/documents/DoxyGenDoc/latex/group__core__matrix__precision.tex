\doxysubsubsection{Detailed Description}
\doxylink{struct_matrix}{Matrix} types with precision qualifiers which may result in various precision in term of ULPs. 

GLSL allows defining qualifiers for particular variables. With Open\+GL\textquotesingle{}s GLSL, these qualifiers have no effect; they are there for compatibility, with Open\+GL ES\textquotesingle{}s GLSL, these qualifiers do have an effect.

C++ has no language equivalent to qualifier qualifiers. So GLM provides the next-\/best thing\+: a number of typedefs that use a particular qualifier.

None of these types make any guarantees about the actual qualifier used.
