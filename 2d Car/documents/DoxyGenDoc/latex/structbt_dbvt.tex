\doxysubsection{Detailed Description}
The \doxylink{structbt_dbvt}{bt\+Dbvt} class implements a fast dynamic bounding volume tree based on axis aligned bounding boxes (aabb tree). This \doxylink{structbt_dbvt}{bt\+Dbvt} is used for soft body collision detection and for the \doxylink{structbt_dbvt_broadphase}{bt\+Dbvt\+Broadphase}. It has a fast insert, remove and update of nodes. Unlike the bt\+Quantized\+Bvh, nodes can be dynamically moved around, which allows for change in topology of the underlying data structure.
