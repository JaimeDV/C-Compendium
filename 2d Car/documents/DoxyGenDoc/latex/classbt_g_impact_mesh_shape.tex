\doxysubsection{Detailed Description}
This class manages a mesh supplied by the bt\+Striding\+Mesh\+Interface interface. 

Set of \doxylink{classbt_g_impact_mesh_shape_part}{bt\+GImpact\+Mesh\+Shape\+Part} parts
\begin{DoxyItemize}
\item Simply create this shape by passing the bt\+Striding\+Mesh\+Interface to the constructor \doxylink{classbt_g_impact_mesh_shape}{bt\+GImpact\+Mesh\+Shape}, then you must call \doxylink{classbt_g_impact_shape_interface_acb26c2d7a2aecabd06b996b72b848492}{update\+Bound()} after creating the mesh
\item You can handle deformable meshes with this shape, by calling \doxylink{classbt_g_impact_mesh_shape_afbf7ae3408bc0ab975c46ed65820a4b2}{post\+Update()} every time when changing the mesh vertices. 
\end{DoxyItemize}
