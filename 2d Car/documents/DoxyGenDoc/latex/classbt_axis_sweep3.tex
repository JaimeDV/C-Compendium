\doxysubsection{Detailed Description}
The \doxylink{classbt_axis_sweep3}{bt\+Axis\+Sweep3} is an efficient implementation of the 3d axis sweep and prune broadphase. It uses arrays rather then lists for storage of the 3 axis. Also it operates using 16 bit integer coordinates instead of floats. For large worlds and many objects, use \doxylink{classbt32_bit_axis_sweep3}{bt32\+Bit\+Axis\+Sweep3} or \doxylink{structbt_dbvt_broadphase}{bt\+Dbvt\+Broadphase} instead. \doxylink{classbt32_bit_axis_sweep3}{bt32\+Bit\+Axis\+Sweep3} has higher precision and allows more then 16384 objects at the cost of more memory and bit of performance.
