\doxysubsection{Detailed Description}
contains definitions and classes for the main sound reproduction loop 

Controls the function \& classes of the game engine window.

Defines implementation of the classes defined in task.

implements scene classes that holds the kernel loop

Implements the grafict funtion using some Sdl2\+Examples code.

implments themethods defined in \doxylink{_message_8hpp}{message.\+hpp}

defines helpfull mat functions for the engine

implements the classes defined in kernel

contains implementations of the classes in \doxylink{_input_manager_8hpp}{input\+Manager.\+Hpp}

Implements the funtión defined in \doxylink{_audio_manager_8hpp}{Audio\+Manager.\+hpp}.

defines the vertex class and the relevant methods

Ensures that some types are allways defined, to ensure older c++ can copilate the code.

defines functions to create a timer class

Defines task classes that the kernel call in every loop.

contains definitions and classes for a math sphere

defines functions to create pseudo randon numbers using xorshift

defines the point class and the relevant methods

Defines and playercontroller class that recives and send messages relevant to the player.

defines the plane class and the relevant methods

Defines and observer class to send messages between player \& engine ~\newline


defines helpfull mat options for the engine and definition for common math types

contains definitions of the classes and macros related to keyboard key events, doesn\textquotesingle{}t contain all only the most usefull for videogame development

Manages the main Input Loop.

Defines entity funtions, cuarently unused.
