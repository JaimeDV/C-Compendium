Get driver-\/specific information about a window.

You must include \doxylink{_s_d_l__syswm_8h}{SDL\+\_\+syswm.\+h} for the declaration of \doxylink{struct_s_d_l___sys_w_minfo}{SDL\+\_\+\+Sys\+WMinfo}.

The caller must initialize the {\ttfamily info} structure\textquotesingle{}s version by using {\ttfamily SDL\+\_\+\+VERSION(\&info.\+version)}, and then this function will fill in the rest of the structure with information about the given window.
