\doxysubsection{Detailed Description}
Additional features not specified by GLSL specification. 

Functions and types that the GLSL specification doesn\textquotesingle{}t define, but useful to have for a C++ program.

GTC extensions aim to be stable with tests and documentation.

Even if it\textquotesingle{}s highly unrecommended, it\textquotesingle{}s possible to include all the extensions at once by including \texorpdfstring{$<$}{<}glm/ext.\+hpp\texorpdfstring{$>$}{>}. Otherwise, each extension needs to be included a specific file.
