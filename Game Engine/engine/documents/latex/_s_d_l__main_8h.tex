This function is defined in \doxylink{_s_d_l__main_8h}{SDL\+\_\+main.\+h}, along with the preprocessor rule to redefine \doxylink{dummy_8cpp_ae66f6b31b5ad750f1fe042a706a4e3d4}{main()} as \doxylink{_s_d_l__main_8h_a35282508ae396e1fd5711ff3d85e08a7}{SDL\+\_\+main()}. Thus to ensure that your \doxylink{dummy_8cpp_ae66f6b31b5ad750f1fe042a706a4e3d4}{main()} function will not be changed it is necessary to define SDL\+\_\+\+MAIN\+\_\+\+HANDLED before including \doxylink{_s_d_l_8h}{SDL.\+h}.
